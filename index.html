Un poco sobre mí

Developer con ojo de diseñadora, diseñadora con mente de developer

Teniendo conocimientos de desarrollo, el diseño puede ser más creativo y razonable. Al poder comprender el diseño, el resultado del desarrollo puede ser más preciso y claro.

Desde que comencé hace casi 10 años, he trabajado para consultoras, para nuevas empresas, como freelancer e incluso para talleres de nuevas tecnologías para niños. He colaborado con personas talentosas para crear productos digitales para uso comercial y de consumo, y para producto propio.

Front end developer

I like to code things from scratch, and enjoy bringing ideas to life in the browser.

Languages I speak:

HTML, Pug, Slim, CSS, Sass, Less

Dev Tools:
